## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/siddharth2019032/study-planner-agent.git
```

Move into the project

```bash
cd study-planner-agent
```

Install dependencies

```bash
agents-cli install
```

Run the project

```bash
agents-cli playground
```
